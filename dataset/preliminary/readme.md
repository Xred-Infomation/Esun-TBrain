# 檔案說明

```plaintext
dataset/preliminary/

│   ├── questions_finance.json          # 財務問題資料集
```

* questions_finance.json：財務問題資料集。
* questions_insurance.json：保險問題資料集。
* pred_retrieve_faq.json：FAQ 預測結果。
* pred_retrieve_finance.json：財務預測結果。
* pred_retrieve_insurance.json：保險預測結果。
